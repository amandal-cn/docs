  <Tab title="For Users">
    ### Quick Start for Hive Users
    
    Follow these steps to start using MCP Hive:

    1. **Create a Workspace**
       - Sign in to MCP Hive
       - Create a new workspace to manage your hives
    
    2. **Add Hives**
       - Browse the marketplace
       - Add desired hives to your workspace
    
    3. **Configure Authorization**
       - Set up authentication for your added hives
       - Configure any required API keys or credentials
    
    4. **Connect to MCP Server**
       - Copy the connection URL from your workspace
       - Use this URL to connect your MCP client

      Check out our demo video to see MCP Hive in action
    </Card>
  </Tab>
  
  <Tab title="For Creators">
    ### Quick Start for Hive Creators
    
    Create your own hive in two ways:

    1. **Using API Specification**
       - Provide your API specification
       - Add metadata (name, description)
       - Configure authentication
       - Set relevant tags
       - Publish your hive
    
    2. **Compose from Existing Tools**
       - Browse available tools
       - Select and combine tools
       - Configure the composition
       - Publish your hive
